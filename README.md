# 差旅搭子 · Travel Buddy

> 面向频繁出差人群的**发票集中管理**桌面软件：从邮件、链接、本地各种方式采集发票，智能解析、按行程与分类归集，一键导出 Excel / PDF 报销单。

---

## 一、为什么需要它

出差党常遇到这些痛点：

- 发票散落在**邮件附件、微信/钉钉链接、本地文件夹**，报销前翻箱倒柜；
- 电子发票（PDF/OFD）字段多，手工抄录**发票号码、金额、税号**费时易错；
- 多段行程混在一起，算不清**每段花了多少、哪些还没报销**；
- 通用报销模板不贴合差旅场景，**Excel 手工统计**效率低。

「差旅搭子」把上述流程收敛到一个本地桌面应用里，数据**留在你自己电脑上**，隐私可控。

---

## 二、核心功能

| 模块 | 能力 |
| --- | --- |
| **多源采集** | ① 本地文件（PDF / OFD / 图片，支持多选批量导入）② 邮件（IMAP 拉取发票附件）③ 链接（粘贴 URL 下载）④ 手工录入 |
| **智能解析** | 从 PDF 文本 / 图片 OCR 文本中正则抽取：发票代码、发票号码、价税合计、税额、不含税金额、开票日期、销售方名称与税号、购买方名称与税号；自动归类（火车/机票/酒店/打车/餐饮等）；解析结果可编辑修正 |
| **集中管理** | 按「行程 / 分类 / 状态」三维组织；关键字搜索、按日期/金额/状态筛选排序；详情查看与编辑；原始附件留存 |
| **仪表盘** | 发票总数、总金额、待报销金额、已报销金额；分类金额占比；近 12 个月趋势；行程报销汇总 |
| **报销导出** | 一键导出 **Excel 报销明细表 + 汇总表**、**PDF 报销单**（中文 CID 字体，无需额外字体文件） |
| **操作手册** | 配套图文操作手册 [`docs/操作手册.md`](docs/操作手册.md)，含 6 张界面截图与 mermaid 流程图 |
| **数据本地化** | SQLite 单文件数据库 + 附件落盘到本地目录，无云端强制上传 |

## 三、技术栈

| 层 | 选型 | 说明 |
| --- | --- | --- |
| 语言 | Python 3.11 | 跨平台、生态丰富 |
| 界面 | **PySide6**（Qt 官方绑定） | 原生桌面体验，跨 Windows / macOS / Linux |
| 数据库 | **SQLite**（嵌入式单文件） | 原方案 H2 依赖 JVM，已替换为零依赖的 SQLite |
| PDF 解析 | pdfplumber | 抽取电子发票文本 |
| 图片 OCR | pytesseract（可选） | 若系统已装 tesseract+中文包则自动启用，否则图片类发票走手工录入 |
| 导出 | openpyxl（Excel）、reportlab（PDF） | 报销单生成 |
| 图表 | matplotlib | 仪表盘统计图表 |

> 说明：本项目最初计划使用 H2（Java 数据库），经评估改为 **SQLite**——同样是嵌入式文件数据库，但纯 Python、无需 JVM，更契合「本地轻量桌面应用」定位。

## 五、安装与运行

### 环境要求
- Python 3.10 及以上
- （可选）tesseract + 中文语言包，用于图片发票 OCR

### 步骤
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动
python main.py
# 或 Linux/macOS 直接：
./run.sh
# Windows：双击 run.bat
```

首次启动会在用户主目录创建数据目录（默认 `~/.chailv_dazi/`），内含 `invoices.db` 与 `attachments/`。可在「设置 / 关于」中查看与打开。

## 七、数据模型（SQLite）

- **categories**：分类（火车票/机票/酒店/打车/餐饮…），含默认状态
- **trips**：差旅行程（名称/目的地/起止日期/预算）
- **invoices**：发票主表（号码/分类/金额/税额/开票日期/购销方/行程/状态/来源）
- **attachments**：发票原始附件（PDF/图片落盘）
- **settings**：键值配置（如邮件服务器、账号）

---

## 八、典型使用流程

1. **新建行程**：出发前建一个行程（如「北京出差 2026-07」）。
2. **导入发票**：
   - 本地积攒的 PDF → 导入向导「本地文件」多选；
   - 邮箱里的行程单 → 「邮件导入」填 IMAP 信息一键拉取；
   - 微信/钉钉收到的链接 → 「链接下载」粘贴即可。
3. **确认与修正**：导入预览中勾选、修正分类与金额，按行程归集。
4. **查看仪表盘**：掌握总金额、待报销额、各分类占比。
5. **导出报销**：月底导出 Excel 明细表交给财务，或打印 PDF 报销单。

## 九、常见问题

- **图片发票识别不到文字？** 需安装 `tesseract` 及中文语言包（`pip install pytesseract` + 系统装 tesseract）。未安装时图片仍会作为附件留存，可手工录入。
- **OFD 格式？** 当前版本暂不支持 OFD 文本抽取，文件会作为附件保留，请手工录入。
- **数据在哪？** 默认 `~/.chailv_dazi/`，可整体复制备份。
- **切换电脑？** 复制整个数据目录即可，数据库为单文件，无外部依赖。
