# 差旅搭子 - 跨平台打包发布指南

## 概述

差旅搭子使用 **PyInstaller** 将 Python 应用打包为原生可执行文件，用户无需安装 Python 和任何依赖，下载安装包后即可直接运行。

支持三个平台：
- **Windows**：`.zip`（解压即用）或 NSIS `.exe` 安装包
- **macOS**：`.dmg`（标准磁盘映像，拖入应用程序）
- **Linux**：`.tar.gz`（解压即用）

## 方式一：GitHub Actions 自动构建（推荐）

### 触发构建

```bash
# 打 tag 并推送，自动触发三平台构建 + Release 发布
git tag v1.0.0
git push origin v1.0.0
```

### 流程说明

1. 推送 `v*` 格式的 tag 后，GitHub Actions 在 Windows / macOS / Linux 三个虚拟机并行构建
2. 每个平台用 PyInstaller 打包，生成对应格式（zip / dmg / tar.gz）
3. 自动创建 GitHub Release，附上三个安装包
4. 用户在 Release 页面直接下载

### 产物

| 平台 | 文件名 | 使用方式 |
|---|---|---|
| Windows | `差旅搭子-windows-x64.zip` | 解压后双击 `差旅搭子.exe` |
| macOS | `差旅搭子-macos.dmg` | 打开 dmg，拖入「应用程序」 |
| Linux | `差旅搭子-linux-x64.tar.gz` | 解压后运行 `./差旅搭子` |

### 手动触发

在 GitHub 仓库 → Actions →「构建跨平台安装包」→ Run workflow，可手动触发构建（不发布 Release）。

## 方式二：本地打包

### 前置条件

- Python 3.10+（需带 `--enable-shared` 编译，系统包默认满足）
- 安装依赖：`pip install pyinstaller -r requirements.txt`

### Windows

```bat
:: 双击执行
scripts\build.bat

:: 或命令行
python -m PyInstaller chailv_dazi.spec --noconfirm
```

产物：`dist\差旅搭子\差旅搭子.exe`

**制作 NSIS 安装包**（需安装 [NSIS](https://nsis.sourceforge.io)）：
```bat
makensis scripts\installer.nsi
```
产物：`dist\差旅搭子_Setup_v1.0.0.exe`

### macOS

```bash
bash scripts/build.sh

# 制作 DMG
hdiutil create -volname "差旅搭子" \
    -srcfolder dist/差旅搭子.app \
    -ov -format UDZO dist/差旅搭子.dmg
```

产物：`dist/差旅搭子.app` 或 `dist/差旅搭子.dmg`

### Linux

```bash
bash scripts/build.sh
```

产物：`dist/差旅搭子/差旅搭子`

## 打包配置说明

### chailv_dazi.spec

核心配置文件，定义了：

- **入口**：`main.py`
- **隐藏导入**：matplotlib 后端、PySide6 QtSvg/QtSvgWidgets、pdfplumber、openpyxl、reportlab
- **资源**：matplotlib 字体数据、`assets/manual/`、`docs/`
- **排除**：tkinter、PyQt5/6、test 模块（减小体积）
- **模式**：`onedir`（目录模式，启动快，避免杀软误报）
- **无控制台**：`console=False`（GUI 应用）
- **平台适配**：
  - macOS → 生成 `.app` 包（BUNDLE）
  - Windows/Linux → 生成目录（COLLECT）

### 体积优化

当前约 400MB，主要来自 PySide6（~150MB）+ matplotlib（~50MB）+ pandas（~50MB）。可进一步优化：

- 使用 `--exclude-module` 排除不需要的 PySide6 模块（QtWebEngine、Qt3D 等）
- 用 UPX 压缩（`upx=True`，但可能触发杀软误报）
- 考虑 Nuitka 替代 PyInstaller（编译为 C，体积更小但构建更慢）

## 常见问题

### macOS 首次打开提示"无法验证开发者"

「系统设置」→「隐私与安全性」→ 点击「仍要打开」。或命令行：
```bash
xattr -cr /Applications/差旅搭子.app
```

### Windows 被 SmartScreen 拦截

点击「更多信息」→「仍要运行」。正式发布需申请代码签名证书。

### Linux 缺少 Qt 依赖

```bash
sudo apt install libgl1 libegl1 libdbus-1-3 libxkbcommon0 libxcb-cursor0
```

### PyInstaller 报 "Python was built without shared library"

使用系统 Python（`/usr/bin/python3`）而非 pyenv 自编译版本，或重新编译 Python 时加 `--enable-shared`。
